🐾 IoT-Based Pet Tracking System (Arduino)

📌 Project Overview

This project contains the Arduino code for the IoT-based pet tracking system. It collects GPS data and sends it to the backend server.

🛠 Technologies Used

- Arduino Nano / ESP32
- GPS Module
- GSM Module

⚙️ Features

- Real-time GPS tracking
- Data transmission using GSM
- Lightweight and efficient device

📡 How It Works

The Arduino device collects location data from the GPS module and sends it to the backend server using a mobile network.
